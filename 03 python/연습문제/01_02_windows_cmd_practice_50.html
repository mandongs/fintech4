Windows CMD 실습문제 50문제 + 정답

디렉토리/파일 관리, 복사/이동, 네트워크 확인, 리다이렉션을 한 번에 연습하는 실습 세트이다.

📂 디렉토리 생성/삭제 📄 파일 생성/삭제/조회 📂 파일 복사/이동 🌐 네트워크/IP 확인 🔄 리다이렉션 활용

📂 디렉토리 생성/삭제

1번

현재 위치에 test 폴더를 생성하시오.

정답 CMD 보기
mkdir test

2번

test 폴더 안에 sub1과 sub2 폴더를 한 번에 생성하시오.

정답 CMD 보기
mkdir test\sub1 test\sub2

3번

test 폴더 안으로 들어가시오.

정답 CMD 보기
cd test

4번

상위 디렉토리로 이동하시오.

정답 CMD 보기
cd ..

5번

절대경로 C:\Windows로 이동하시오.

정답 CMD 보기
cd C:\Windows

6번

상대경로를 이용해 현재 폴더의 sub1 디렉토리로 이동하시오.

정답 CMD 보기
cd ..\test\sub1

7번

rd 명령어로 빈 sub2 폴더를 삭제하시오.

정답 CMD 보기
rd ..\sub2

8번

rmdir /s /q로 test 폴더와 안의 모든 내용을 삭제하시오.

정답 CMD 보기
cd c:\
rmdir /s /q test
주의 이 명령은 하위 폴더/파일까지 강제 삭제하므로 경로를 반드시 확인해야 한다.

9번

현재 디렉토리 경로를 확인하시오.

정답 CMD 보기
cd

10번

2번의 작업을 다시 실행하고 test 디렉토리 구조를 트리 형태로 출력하시오.

정답 CMD 보기
mkdir test\sub1 test\sub2
tree test

📄 파일 생성/삭제/조회

11번

test 디렉토리 안에 a.txt를 생성하고 Hello를 저장하시오.

정답 CMD 보기
echo Hello > a.txt

12번

a.txt 파일에 World를 추가하시오.

정답 CMD 보기
echo World >> a.txt

13번

a.txt 내용을 화면에 출력하시오.

정답 CMD 보기
type a.txt

14번

copy con b.txt로 직접 입력하여 b.txt를 만드시오. (Ctrl+Z 후 Enter)

정답 CMD 보기
copy con b.txt

15번

a.txt의 이름을 c.txt로 변경하시오.

정답 CMD 보기
rename a.txt c.txt

16번

c.txt 파일을 삭제하시오.

정답 CMD 보기
del c.txt

17번

d.txt를 생성하고 CMD TEST를 넣으시오.

정답 CMD 보기
echo CMD TEST > d.txt

18번

type 명령어로 d.txt 내용을 확인하시오.

정답 CMD 보기
type d.txt

19번

dir로 현재 폴더의 파일과 디렉토리 목록을 확인하시오.

정답 CMD 보기
dir

20번

dir /w 옵션을 사용해 가로로 출력하시오.

정답 CMD 보기
dir /w

📂 파일 복사/이동

21번

d.txt를 복사해 e.txt를 만드시오.

정답 CMD 보기
copy d.txt e.txt

22번

e.txt를 sub1 폴더로 이동하시오.

정답 CMD 보기
move e.txt sub1\

23번

sub1\e.txt를 복사해 f.txt를 만드시오.

정답 CMD 보기
copy sub1\e.txt f.txt

24번

f.txt를 상위 디렉토리로 이동하시오.

정답 CMD 보기
move f.txt ..\

25번

xcopy로 sub1 폴더를 backup으로 하위 포함 복사하시오.

정답 CMD 보기
xcopy sub1 backup /e

26번

robocopy로 sub1 폴더를 backup2에 복사하시오.

정답 CMD 보기
robocopy sub1 backup2 /e

27번

이동한 파일 sub1\e.txt를 삭제하시오.

정답 CMD 보기
del sub1\e.txt

28번

d.txt를 복사해 sub1\g.txt를 생성하시오.

정답 CMD 보기
copy d.txt sub1\g.txt

29번

dir로 sub1 폴더의 파일 목록을 확인하시오.

정답 CMD 보기
dir sub1

30번

tree /f로 현재 폴더 구조와 파일까지 출력하시오.

정답 CMD 보기
tree /f

🌐 네트워크/IP 확인

31번

IP 주소 정보를 확인하시오.

정답 CMD 보기
ipconfig

32번

네트워크 어댑터의 모든 세부 정보를 확인하시오.

정답 CMD 보기
ipconfig /all

33번

Google 서버에 ping을 보내 연결을 테스트하시오.

정답 CMD 보기
ping www.google.com

34번

현재 컴퓨터 이름을 확인하시오.

정답 CMD 보기
hostname

35번

현재 사용자 이름을 출력하시오.

정답 CMD 보기
echo %username%

36번

현재 컴퓨터 이름을 출력하시오.

정답 CMD 보기
echo %computername%

37번

현재 시간과 날짜를 확인하시오.

정답 CMD 보기
time
date

38번

Windows 버전을 확인하시오.

정답 CMD 보기
ver

39번

시스템 정보를 확인하시오.

정답 CMD 보기
systeminfo

40번

현재 경로를 확인한 뒤, 그 결과를 화면과 동시에 파일로 저장하시오.

정답 CMD 보기
cd
cd > path.txt
참고 원문 그대로라 2줄로 제공되어 있다. “화면과 동시에”까지 완벽히 하려면 tee 같은 도구가 필요하지만, 기본 CMD 범위에서는 보통 이렇게 파일 저장을 연습한다.

🔄 리다이렉션 활용

41번

현재 디렉토리 목록을 files.txt에 저장하시오.

정답 CMD 보기
dir > files.txt

42번

IP 정보를 ip.txt에 저장하시오.

정답 CMD 보기
ipconfig > ip.txt

43번

files.txt에 Test라는 문장을 추가하시오.

정답 CMD 보기
echo Test >> files.txt

44번

files.txt에 저장된 내용을 확인하시오.

정답 CMD 보기
type files.txt

45번

디렉토리 구조를 tree.txt에 저장하시오.

정답 CMD 보기
tree > tree.txt

46번

ping 결과를 ping.txt에 저장하시오.

정답 CMD 보기
ping www.google.com > ping.txt

47번

빈 파일 empty.txt를 생성하시오.

정답 CMD 보기
type nul > empty.txt

48번

copy con notes.txt로 notes.txt를 만들고 내용을 입력하시오. (Ctrl+Z로 종료)

정답 CMD 보기
copy con notes.txt

49번

log.txt에 디렉토리 목록을 누적 저장하시오.

정답 CMD 보기
dir >> log.txt

50번

log.txt에 최종 저장된 내용을 확인하시오.

정답 CMD 보기
type log.txt
학습 팁 41~50번은 “결과를 파일로 남기기(로그)”를 연습하는 파트이다. >는 덮어쓰기, >>는 추가 저장이다.
